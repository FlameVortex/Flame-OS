## 🚀 Key Features

*   ✅ **Auto-Installation:** Installs all required packages (Python, Git, PHP, ZSH, etc.) in one click.
*   ✅ **Premium Branding:** Generates a professional 3D ASCII banner with your own name.
*   ✅ **Smart Shell:** Configures ZSH with **Auto-suggestions** and **Syntax Highlighting**.
*   ✅ **Fast & Lite:** Optimized to run smoothly on any Android device.

---

## 📥 Installation Guide

Copy and paste the code below into your Termux terminal to start the setup:

```bash
yes | pkg update && yes | pkg upgrade && pkg install python git -y && pkg reinstall python -y && hash -r && rm -rf Flame-OS && git clone https://github.com/FlameVortex/Flame-OS && cd Flame-OS && python3 setup.py
```
